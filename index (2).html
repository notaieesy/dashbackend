Hapus semua data yang tersimpan di browser (localStorage). Data di Google Sheets tidak terpengaruh. Session login tetap aman.
Cara setup (pertama kali)
  1. Buka Google Sheets baru → klik Extensions → Apps Script
  2. Copy-paste isi file Code.gs yang kamu download ke editor Apps Script
  3. Jalankan fungsi setup() sekali untuk membuat semua sheet otomatis
  4. Klik Deploy → New deployment → Web app
  5. Set Execute as: Me dan Who has access: Anyone → Deploy
  6. Copy URL yang muncul, paste di field di atas, klik Save URL
  7. Klik Push to Sheets untuk kirim data awal ke Sheets
